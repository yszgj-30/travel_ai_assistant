\# 🌍 智能出行助手 AI

基于大模型 + 工具调用的智能旅行助手系统，集成\*\*天气查询、行程规划、文件保存\*\*三大核心功能，提供一站式旅行决策支持。



\---



\## ✨ 项目亮点

\- 📍 \*\*实时天气查询\*\*：对接高德地图天气API，支持全国所有城市实时天气数据

\- 🚗 \*\*智能行程规划\*\*：结合天气情况生成结构化出行方案，支持多路线对比

\- 📄 \*\*自动文件保存\*\*：一键将旅行计划、文本内容保存为本地文件，支持带时间戳命名

\- 🤖 \*\*多轮对话交互\*\*：大模型驱动上下文理解，支持自然语言提问

\- 🎨 \*\*现代化前端\*\*：基于 Streamlit 构建的简洁对话界面，交互友好



\---



\## 🛠️ 技术栈

| 模块 | 技术/工具 |

| :--- | :--- |

| 后端服务 | Python + FastAPI |

| 大模型 | 通义千问 API |

| 天气服务 | 高德地图 Web 服务 API |

| 前端界面 | Streamlit |

| 部署运行 | Uvicorn |
